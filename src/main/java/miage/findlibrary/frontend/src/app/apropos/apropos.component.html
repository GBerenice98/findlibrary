Bienvenue sur FindYourBiblio !


FindYourBiblio est une application de recherche de bibliothèques logicielles cibles dans le cadre d'une migration. Cette application créé dans le but de faciliter la lecture des graphes de migrations, utilise les données issues des travaux de Cédric TeyTon et al. expliqués dans le document https://www.labri.fr/perso/xblanc/data/thesis/TEYTON.pdf
