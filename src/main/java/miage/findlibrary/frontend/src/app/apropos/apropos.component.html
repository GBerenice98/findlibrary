Bienvenue sur FindYourBiblio !


FindYourBiblio est une application de recherche de bibliothèques logicielles cibles dans le cadre d'une migration. Cette application est créée dans le but de faciliter la lecture des graphes de migrations. Elle utilise les données issues des travaux de Cédric TeyTon et al. expliqués dans le document Gestion des bibliothèques tierces dans un contexte de maintenance logicielle.
